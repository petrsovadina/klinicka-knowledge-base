## 6. Metodika Extrakce

### 6.1 Manuální Extrakce (Týden 1–2)
1. Stáhnout PDF dokument
2. Přečíst a identifikovat klíčové části
3. Extrahovat pravidla, výjimky, rizika
4. Strukturovat do JSON formátu
5. Validovat a propojit s existujícími jednotkami

**Čas na jednotku**: 15–20 minut  
**Kapacita**: 3–4 jednotky/hodina

### 6.2 LLM-asistovaná Extrakce (Týden 3+)
1. Stáhnout PDF dokument
2. Extrahovat text pomocí `pdftotext` nebo `pypdf`
3. Předat text LLM (GPT-4o / Claude 3.5) s promptem
4. LLM vygeneruje strukturovaný JSON
5. Manuální review a validace
6. Propojení s existujícími jednotkami

**Čas na jednotku**: 5–10 minut  
**Kapacita**: 6–12 jednotek/hodina

### 6.3 Prompt Template pro LLM
```
Jsi expert na české zdravotnictví a úhradové mechanismy. Tvým úkolem je extrahovat znalostní jednotky z následujícího textu.

TEXT:
{text}

ÚKOL:
1. Identifikuj všechna pravidla, výjimky, rizika, podmínky a anti-patterny
2. Pro každou znalostní jednotku vytvoř JSON objekt dle schématu
3. Zajisti, aby každá jednotka měla:
   - Jasný název (title)
   - Detailní popis (description)
   - Strukturovaný obsah (content)
   - Zdroj (source)
   - Aplikovatelnost (applicability)

SCHÉMA:
{json_schema}

VÝSTUP:
Vrať pouze validní JSON objekty, jeden na řádek (JSON Lines formát).
```

## 8. Nástroje a Infrastruktura

### 8.1 Potřebné Nástroje
- **PDF extrakce**: `pdftotext`, `pypdf2`, `pdfplumber`
- **LLM API**: OpenAI GPT-4o, Anthropic Claude 3.5 Sonnet
- **Validace**: `jsonschema`, custom Python skripty
- **Version control**: Git, GitHub

### 8.2 Skripty k Vytvoření
1. `download_sources.py` – Stahování PDF dokumentů

4. `validate_units.py` – Validace znalostních jednotek
